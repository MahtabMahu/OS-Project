# OS-Project

Here is my simple Restaurant Ordering System project using shell scripting(bash) languages.

For run the file in Linux Operating System. 
Go to the terminal and use the command: chmod u+x Mahtab.sh
Then Command: ./Mahtab.sh


Features :

Refined Interface
User Guidance
Input Validation
Clear Prompts
Order Confirmation
Error Handling
File Output
Item List
Dynamic Pricing
Order Reset
Interactive Feedback
Confirmation Checks
Detailed Receipt
Enhanced Usability
Quantity Support
Flexible Navigation
Clean Formatting
Friendly Flow
Multiple Options
Easy Cancellation
